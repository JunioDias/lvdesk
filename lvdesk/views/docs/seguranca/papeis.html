Introdução

O LV Desk possui 3 papéis pré-definidos: Administrador, Coordenador e Atendente. Cada papel vem com permissões para executar as tarefas mostradas nas tabelas de cada módulo do sistema.

Ao vincular um usuário ou grupo de usuários a um determinado papel, aquele objeto receberá todos os privilégios que as permissões concedidas lhe proporcionarão.
